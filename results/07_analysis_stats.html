Statistics: t-test and p-values

Using a t-test to check whether or not the KO and baseline samples are different for each cell line.

KO_OVCAR3 <- OVCAR3_mock_or_ko |> 
  filter(MOCK_or_KO == "KO") |>
  select(OVCAR3)

MOCK_OVCAR3 <- OVCAR3_mock_or_ko |> 
  filter(MOCK_or_KO == "MOCK") |> 
  select(OVCAR3)

#t-test for OVCAR3
broom::tidy(t.test(KO_OVCAR3, MOCK_OVCAR3), paired = TRUE)|> 
  pull(p.value)
## [1] 0.7957103
KO_SKOV3 <- SKOV3_mock_or_ko |> 
  filter(MOCK_or_KO == "KO") |> 
  select(SKOV3)

MOCK_SKOV3 <- SKOV3_mock_or_ko |> 
  filter(MOCK_or_KO == "MOCK") |> 
  select(SKOV3)

#t-test for SKOV3
broom::tidy(t.test(KO_SKOV3, MOCK_SKOV3), paired = TRUE)|> 
  pull(p.value)
## [1] 0.06881636

The p-value is 0.79 for OVCAR3. This means that there are no statistical significant difference between the values of KO vs baseline (MOCK) in this cell line.

The p-value is 0.0688 for SKOV3. This means that there are no statistical significant difference between the values of KO vs baseline (MOCK) in this cell line.

Significant difference between KO and baseline in the cell lines for GDF15 gene

The GDF15 gene has been shown to be highly induced in ovarian cell cancer, this gene is investigated for our cell lines and their KOs.

gdf15 <- data |>
